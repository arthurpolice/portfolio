Professional Projects

Oxford Scholastica and Oxbright

This is a booking form project made under the mentorship of a senior Freelancer for the summer school Oxford Scholastica Academy and their sister online school Oxbright.

The project consists of a dual domain website, detecting which school the booking is to be made for and adapting accordingly. As it stands, it is a purely front-end project, using bubble.io as its RESTful API backend, though the client plans to move to a custom made backend in the future.

This project was built using mainly React, Next.js and Typescript. We also integrated various other services, such as Stripe, Google Tag Manager, Referral Hero, Zapier and others.

Zuppy

This is a fintech project, dedicated to tracking online sales and automatically creating tax invoices.

I worked as a junior in this project for about half a year, during which time I worked on multiple fronts, such as creating tutorials for users, integrating our services with various payment processing platforms and performing the necessary frontend and backend changes for those, creating an error logging system for our application that fired emails to the clients and messages on our slack server, among other varied tasks pertaining a new project (UI, UX, planning new features).

This project was built using React, Next.js, Typescript, tRPC and Tailwind. Of particular interest is the use of tRPC instead of a traditional REST API approach to frontend and backend communication. This provided end to end type safety, which really boosted productivity, since changes in the database immediately reflected on the code and warned us of possible issues.
